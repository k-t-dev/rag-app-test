# ネイル会社サンプルマニュアル

このWebAppは初回起動時に以下の社内マニュアルをSQLiteへ投入します。

- 接客基本マニュアル: 全社公開
- ジェルネイル施術マニュアル: 全社公開
- 衛生管理マニュアル: 全社公開
- 予約・キャンセル対応: 全社公開
- クレーム一次対応: マネージャー限定
- スタッフ教育マニュアル: マネージャー限定
- 本社KPI・売上分析: 本社限定
- 渋谷店ローカル運用: 渋谷店限定
- 新宿店ローカル運用: 新宿店限定
- 個人情報事故対応: 管理者限定

Web UIのマニュアル管理画面から本文、公開範囲、対象支店、ステータスを更新できます。
更新後は「再Embedding」を実行すると検索対象のチャンクが新しい版へ差し替わります。
